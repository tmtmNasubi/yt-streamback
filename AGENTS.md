# Codex エージェント運用ガイド

このリポジトリで作業する Codex は、以下のルールを必ず守ってください。

## 言語ポリシー
- ユーザーとの対話は日本語を優先する。
- Pull Request のタイトル・本文・要約は日本語で作成する。
- レビューコメント（指摘・提案・確認事項）も日本語で記述する。

## 開発ワークフロー
- 変更前に影響範囲を確認し、必要に応じてテストを実行する。
- 変更内容は最小限かつ明確にし、関連しない変更を含めない。
- 変更後は、実行したコマンドと結果を要約に含める。

## 変更サマリの書式
- サマリは箇条書きで簡潔に記載する。
- 主要な変更点、テスト結果、既知の制約を分けて記載する。
